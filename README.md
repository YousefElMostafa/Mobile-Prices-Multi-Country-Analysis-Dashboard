# Mobile Prices Multi-Country Analysis Dashboard

## Project Name
Mobile Prices Multi-Country Analysis & Forecasting Dashboard

## Project Idea
This project aims to analyze and compare mobile phone prices across Egypt, Saudi Arabia, and UAE using real e-commerce data.
The project includes data collection, data cleaning, exploratory data analysis, and price forecasting.
An interactive dashboard will be built using Power BI to visualize insights and support decision-making.

## Project Plan

### Week 1 – Data Cleaning & Modeling
- Collecting data from e-commerce platforms
- Data cleaning and preprocessing using Python
- Building SQL database
- Designing data model

### Week 2 – Data Analysis
- Average price per country
- Brand comparison across countries
- Price distribution analysis
- Identifying cheapest and most expensive models
- Category segmentation (Budget / Mid-range / Flagship)

### Week 3 – Forecasting
- Time-series analysis
- Price trend forecasting
- Evaluating forecasting models

### Week 4 – Dashboard & Final Presentation
- Building interactive Power BI dashboard
- Final report preparation
- Final presentation

## Tools Used
- Python (Pandas, Matplotlib, Scikit-learn)
- SQL
- Power BI
